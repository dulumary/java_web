
* input 과 검색 버튼이 있다. 
* 검색 버튼을 눌렀을떄, input 내용에 금칙어가 포함되어 있으면 alert을 통해서 메시지를 출력하세요. 


> 입력 예시 


> 출력 예시 

