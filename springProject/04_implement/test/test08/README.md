# 1. 글 삭제 

* 로그인 된 사용자가 쓴 글인 경우에만 ...(More 버튼) 노출한다.
* 버튼을 클릭하면 삭제 모달창을 띄운다.
* 글 삭제시 수행할 것 
  * post 제거
  * 이미지 파일 제거
  * like 제거
  * 댓글들 제거
